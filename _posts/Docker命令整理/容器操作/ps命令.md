<br/>
<font size=4>**列出容器**</font>
<br/>
<br/>
<code> docker ps [OPTIONS]<br/>
-a, --all=false: 显示所有的容器，包括未运行的<br/>
-l, --all=false: 显示最近创建的容器<br/>
-n, --all=false: 列出最近创建的n个容器<br/>
--no-trunc=false: 不截断输出<br/>
-q: 静默模式，只显示容器编号<br/>
-s: 显示总的文件大小<br/>
</code>
